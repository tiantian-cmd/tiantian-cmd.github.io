小西瓜搭建网站方法

本次讲的是用集成环境搭建一个网站
phpstudy下载地址官网
wordpress下载

大家要注意这两个工具的版本问题,还有就是这个集成环境

若本地装好了mysql，则要将mysql的my.ini文件删除,因为这会影响phpstudy里的mysql无法运行，或输入代码在cmd命令行中

解压wordpress压缩包，将’wordpress’文件夹里面的所有文件放到phpstudy的’WWW’文件夹中；

在浏览器地址栏输入以下地址就可以看到网站开始搭建：127.0.0.1，接下来填写一些基本的信息，主要的是在数据库的文本框中填写步骤2中创建的数据库名称，其它就没问题了，到此就ok了。
接下来我要搞一下外网了，当然你也可以自己搭建一个服务器

其实node.js也可以搭建服务器。就是我不会。
WordPress的运行环境是很传统的Apache + MySQL + PHP，其中PHP用的是PHP5，因此部署过程中应该不会有很难解决的问题，只不过步骤可能会比较繁琐。除此以外就是调试的配置需要摸索一阵子。

一开始我曾想过花钱买MAMP PRO来搭建WordPress的本地开发环境（MAMP免费版本只支持PHP7），图省事。但后来还是决定自己折腾，省下这笔钱。
